Vision Gauge
엣지 AI 카메라 데모

카메라 보드가 클라우드 없이 직접 사물을 인식하고, 판단 결과만 무선으로 보내 OLED와 대시보드에 실시간 표시하는 온디바이스 비전 AI 프로젝트.

Nicla Vision Edge Impulse BLE ESP32 · OLED Streamlit 고등학교 SW·AI 융합수업
01

프로젝트 요약

"AI가 사물을 알아본다"를 직접 학습시키고, 보드에 올리고, 눈으로 확인하는 프로젝트입니다.

학생이 카메라로 사물 이미지를 모아 Edge Impulse에서 분류 모델을 학습하고, 이를 Nicla Vision 보드에 직접 탑재합니다. 인터넷 연결 없이 보드 혼자 추론하며(온디바이스 AI), 인식 결과(클래스·확신도)만 블루투스(BLE)로 ESP32 제어 노드에 보내 OLED에 게이지로 표시합니다. 동시에 USB로 PC에 연결해 Streamlit 대시보드에서 카메라가 보는 영상과 인식 결과를 실시간으로 확인합니다.
